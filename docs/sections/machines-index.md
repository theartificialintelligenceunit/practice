## Initial Set-up

Switch-on the new machine and proceed as guided, <b>but</b>

<ul>
  <li><b>Do not select Microsoft Account.</b>  Instead, select the <b>School/Business</b> option, thereafter select <b>other domains</b> option.</li>
  <li>Install and set up the security software.</li>
  <li>Hard Drive Partitioning: Consider partitioning the C drive into two drives; such that the second drive is for development purposes.</li>
</ul>
